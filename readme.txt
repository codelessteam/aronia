== Description ==

Aronia is a powerful, flexible, and modern theme for your business website. Theme comes wtih Elementor Builder and compatible with WPbakery Visual Composer, this theme is an outstanding choice for basically any kind of company or freelancer site. Bringing a unlimited customization with WordPress Customizer opportunities, Aronia is great if you want to start a website from scratch and it's even greater if you want to use a premade Elementor layout from our growing collection. 

== Installation ==

1. In your admin panel, go to Appearance > Themes and click the Add New button.
2. Click Upload Theme and Choose File, then select the theme's .zip file. Click Install Now.
3. Click Activate to use your new theme right away.
